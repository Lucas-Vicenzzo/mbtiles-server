mbtiles-server
==============

Servidor MBTiles para o projeto Meu IFC

1. nvm install 8.0;
2. nvm use 8.0;
3. npm install express;
4. npm install mbtiles;
5. node server.js map.mbtiles PORTA[3001];

1. `npm install`
1. `node server.js TILEFILE [PORT]`
